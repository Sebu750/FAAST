# Professional Blog Editor Guide

## Overview

The Adorzia blog editor has been upgraded from a plain-text Markdown editor to a **professional rich text editor** powered by TipTap. This guide covers all the features available to create polished, branded blog content.

---

## What's New

### Before (Markdown Editor)
- Plain textarea requiring Markdown syntax knowledge
- Manual formatting with `#`, `**`, `*`, etc.
- No visual feedback while editing
- Limited image controls
- Basic preview

### After (Rich Text Editor)
- **WYSIWYG** (What You See Is What You Get) interface
- Visual formatting toolbar
- Real-time preview
- Advanced image management
- Branded content blocks
- Professional styling

---

## Editor Features

### 1. Text Formatting

#### Headings
- **H1**: Main title (use once per post)
- **H2**: Section headings
- **H3**: Subsection headings  
- **H4**: Minor headings

**How to use:** Click the H1-H4 buttons in the toolbar, then type your heading.

#### Text Styles
- **Bold** (B): Emphasize important text
- **Italic** (I): Subtle emphasis or titles
- **Underline** (U): Strong emphasis
- **Strikethrough** (S): Deleted or outdated text

**How to use:** Select text, then click the style button.

#### Text Color
- Click the color picker to choose a custom text color
- Use brand colors for emphasis
- Click "Reset" to return to default color

#### Highlight
- Add yellow background highlight to text
- Great for callouts or important notes

---

### 2. Text Alignment

Control how text aligns on the page:

- **Align Left**: Default alignment (professional, readable)
- **Align Center**: Headings, pull quotes, CTAs
- **Align Right**: Special elements, signatures
- **Justify**: Formal, newspaper-style alignment

**How to use:** Place cursor in paragraph, click alignment button.

---

### 3. Lists

#### Bullet Lists (Unordered)
- Use for non-sequential items
- Great for features, benefits, key points

#### Numbered Lists (Ordered)
- Use for step-by-step instructions
- Rankings, priorities, sequences

**How to use:** Click list button, then type items. Press Enter for new items.

---

### 4. Blockquotes & Code

#### Blockquotes
- Highlight quotes from interviews
- Emphasize important statements
- Visual distinction from regular text

**How to use:** Click quote icon, type quote text.

#### Code Blocks
- Display code snippets
- Technical content
- Preserves formatting

**How to use:** Click code icon, paste code.

---

### 5. Images

#### Insert Image
**Method 1: URL Input**
1. Click image icon in toolbar
2. Enter image URL
3. Press Enter

**Method 2: Upload (Coming Soon)**
- Drag & drop directly into editor
- Paste from clipboard
- Upload from media library

#### Image Controls (Coming Soon)
- Size: Small (25%), Medium (50%), Large (75%), Full-width (100%)

- Captions
- Alt text for accessibility

---

### 6. Links

**How to add a link:**
1. Select text you want to link
2. Click link icon in toolbar
3. Enter URL in modal
4. Click "Apply"

**Link features:**
- Opens in same tab by default
- Branded gold color (#bb9457)
- Hover underline effect

---

### 7. Horizontal Rules

Visual separators between content sections.

**How to use:** Click horizontal rule icon.

---

### 8. Undo/Redo

- **Undo** (←): Revert last action
- **Redo** (→): Reapply undone action

**Keyboard shortcuts:**
- Windows/Linux: `Ctrl+Z` / `Ctrl+Y`
- Mac: `Cmd+Z` / `Cmd+Y`

## Writing Workflow

### 1. Start a New Post

1. Navigate to Admin Dashboard → Blog Posts → New Post
2. Enter title (auto-generates URL slug)
3. Add excerpt (brief description for listings)

### 2. Upload Images

1. **Banner Image**: Article header (16:9 ratio recommended)
2. **Featured Image**: Card thumbnail (square ratio)
3. **Inline Images**: Within content via toolbar

### 3. Write Content

1. Start typing in the editor
2. Use toolbar to format text
3. Add headings for structure
4. Insert images to break up text
5. Use lists for scannable content

### 4. Format & Style

1. Apply text colors for emphasis
2. Add blockquotes for interviews
3. Insert pull quotes for key statements
4. Use callout boxes for important notes
5. Add CTA buttons for conversions

### 5. Preview

1. Click "Preview" button in header
2. Review how post will appear to readers
3. Check formatting, images, links
4. Return to "Edit" mode for changes

### 6. Save & Publish

**Save Draft:**
- Click "Save Draft" button
- Post saved but not visible to public
- Can edit later

**Publish:**
- Click "Publish" button
- Post immediately visible to public
- Can still edit after publishing

**Schedule:**
- Set future date/time in sidebar
- Post auto-publishes at scheduled time

## Best Practices

### Content Structure

✅ **Do:**
- Use one H1 per post (the title)
- Break content with H2/H3 headings every 200-300 words
- Use short paragraphs (2-4 sentences)
- Add images every 300-400 words
- Include bullet/numbered lists for scannable content

❌ **Don't:**
- Skip heading levels (H1 → H3)
- Write walls of text
- Use too many colors/styles
- Forget to add alt text to images

### Images

✅ **Do:**
- Use high-quality images (min 1200px wide)
- Optimize file size (< 500KB)
- Add descriptive alt text
- Use consistent aspect ratios

❌ **Don't:**
- Stretch or distort images
- Use copyrighted images without permission
- Forget to credit photographers

### Links

✅ **Do:**
- Use descriptive link text ("Read the full report" not "Click here")
- Link to authoritative sources
- Open external links in new tabs

❌ **Don't:**
- Stuff content with too many links
- Use generic "click here" text
- Link to untrusted sources

## Migration from Markdown

Existing Markdown blog posts have been automatically converted to HTML format. The conversion handles:

- Headings (# → H1, ## → H2, etc.)
- Bold/Italic text
- Images
- Links
- Lists
- Blockquotes

**Note:** For best results, review converted posts and apply branded styling (pull quotes, callouts, etc.) to enhance readability.

## Troubleshooting

### Editor not loading?
- Refresh the page
- Clear browser cache
- Check internet connection

### Images not displaying?
- Verify image URLs are correct
- Check image file format (JPG, PNG, WebP supported)
- Ensure images are uploaded to Supabase storage

### Formatting not applying?
- Select text first, then apply formatting
- Some formats only work on selected text
- Try clicking in the paragraph first

### Changes not saving?
- Check for unsaved changes indicator
- Click "Save Draft" manually
- Verify you're logged in
